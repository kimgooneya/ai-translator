# About Contributions

This is a personal project. The source code is publicly available under the [MIT License](LICENSE) for transparency and educational purposes.

## No External Pull Requests

I'm currently **not accepting pull requests** or external contributions to this repository. 

You are welcome to:
- **Fork** the repository for your own use
- **Modify** the code under the terms of the MIT License
- **Use** it as a reference or learning resource
- **Open issues** for bug reports or feature suggestions

However, please do not submit pull requests expecting them to be merged.

## Found a Bug?

If you find a bug, please [open an issue](https://github.com/kimgooneya/ai-translator/issues) with:
- Description of the problem
- Steps to reproduce
- Expected vs actual behavior
- Browser and OS information

Issue reports are appreciated and help improve the project.
